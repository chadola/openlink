# 🐍 贪吃蛇游戏 (Snake Game) - 标准版

一个功能丰富的基于 HTML5 Canvas 的经典贪吃蛇网页游戏。

## 📁 项目结构

```
snake_game/
├── index.html          # 主页面
├── css/
│   ├── style.css       # 主样式文件
│   └── themes.css      # 主题样式文件
├── js/
│   ├── game.js         # 游戏核心逻辑
│   ├── controller.js   # 输入控制（键盘/触屏/手势）
│   ├── storage.js      # 数据存储（最高分/排行榜/成就）
│   └── effects.js      # 特效处理（粒子/音效）
├── assets/
│   ├── sounds/         # 音效文件（可选）
│   └── images/         # 图片资源（可选）
├── snake_game.html     # 原版单文件版本
├── test.txt            # 测试文件
└── README.md           # 项目说明文档
```

## 🎮 游戏特性

### 核心功能
- ✨ 霓虹风格界面设计
- 🎯 多种控制方式（键盘/触屏按钮/手势滑动）
- 💾 本地最高分存储
- 🚀 随得分增加难度递增

### 标准版新增功能
- 🎮 **三种游戏模式**
  - 经典模式：传统贪吃蛇玩法
  - 计时模式：60 秒内尽可能得分
  - 无尽模式：持续挑战极限

- 🎚️ **三种难度等级**
  - 简单：速度慢，适合新手
  - 普通：标准速度
  - 困难：速度快，挑战性高

- 🎨 **四种视觉主题**
  - 霓虹（默认）：经典绿色霓虹风格
  - 暗夜：紫色神秘风格
  - 复古：绿色终端风格
  - 海洋：蓝色清新风格

- 🏆 **成就系统**
  - 10 种不同成就
  - 解锁进度追踪
  - 成就解锁特效

- 📊 **排行榜**
  - 本地前 10 名记录
  - 显示游戏模式和日期

- ⚙️ **设置选项**
  - 音效开关
  - 音乐开关
  - 震动反馈
  - 网格显示

- 🎉 **特效增强**
  - 粒子爆炸效果
  - 音效反馈（Web Audio API）
  - 震动反馈（移动设备）
  - 屏幕闪烁效果

- 🍎 **特殊食物系统**
  - 金色奖励食物：+50 分
  - 蓝色减速食物：暂时减速
  - 粉色加速食物：暂时加速 + 额外分数

## 🚀 使用方法

### 快速开始
1. 用浏览器打开 `index.html`
2. 选择游戏模式、难度和主题
3. 点击「开始游戏」按钮

### 控制方式

#### 键盘控制（桌面端）
| 操作 | 按键 |
|------|------|
| 向上 | ↑ / W |
| 向下 | ↓ / S |
| 向左 | ← / A |
| 向右 | → / D |
| 暂停 | 空格 / Esc |

#### 触屏控制（移动端）
- 方向按钮：点击屏幕方向键
- 手势滑动：在游戏区域滑动控制方向

## 🎯 游戏规则

### 经典模式
- 吃到食物：+10 分（根据难度有倍数）
- 撞墙或撞自己：游戏结束
- 每吃一个食物，蛇的速度会略微加快

### 计时模式
- 60 秒倒计时
- 时间内尽可能获得高分
- 时间到游戏结束

### 无尽模式
- 持续游戏直到失败
- 挑战最高分记录

## 🔧 开发说明

### 模块结构
- `game.js`: 游戏主逻辑，包含游戏循环、渲染、状态管理
- `controller.js`: 输入处理，支持键盘、触屏按钮、手势滑动
- `storage.js`: 数据持久化，管理最高分、排行榜、成就、设置
- `effects.js`: 视觉效果和音效，粒子系统、Web Audio 音效

### 扩展开发
1. 添加新主题：在 `themes.css` 中添加新的主题类
2. 添加新成就：在 `storage.js` 的 `defaultAchievements` 数组中添加
3. 添加新模式：在 `game.js` 中扩展游戏逻辑
